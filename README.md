## âš™ï¸ HOW IT WORKS

1. **DATA GENERATION**
   - SYNTHETIC MATERIAL DATA GENERATED USING UNIFORM DISTRIBUTIONS WITHIN VERIFIED EXPERIMENTAL RANGES.

2. **EXPLORATORY DATA ANALYSIS**
   - PEARSON CORRELATION AND SCATTER MATRICES REVEAL INITIAL RELATIONSHIPS BETWEEN DIELECTRIC CONSTANTS AND FREQUENCY WITH AC CONDUCTIVITY.

3. **NORMALIZATION**
   - MIN-MAX SCALERS ARE APPLIED TO BOTH INDEPENDENT FEATURES AND THE MICROSCOPIC TARGET TO ENSURE STABILITY ACROSS DISTANCE-BASED MODELS (KNN, SVR).

4. **MODEL TRAINING**
   - MULTIPLE REGRESSORS COMPETE. ENSEMBLE METHODS LIKE RANDOM FOREST AND GRADIENT BOOSTING ARE EVALUATED AGAINST BASELINE LINEAR MODELS.

5. **POST-TRAINING DIAGNOSTICS**
   - RÂ², RMSE, MAE, AND RESIDUAL PLOTS CONFIRM MODEL ACCURACY.

6. **PHYSICS VALIDATION (XAI)**
   - SHAP VALUES AND ICE PLOTS EXPLAIN THE EXACT MARGINAL EFFECT OF EACH FEATURE, CORROBORATING EXPERIMENTAL INSIGHTS.

## ðŸ”„ ANALYTICAL DECISION FLOW
